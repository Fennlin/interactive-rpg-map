# Interactive RPG World Map

## Description
This project is a personal portfolio piece that brings together my hobby and passion for RPG worlds.

It is a custom interactive map where locations can be explored through pins, filters, and lore panels. The goal is to present worldbuilding content in a simple and immersive way.

## Website
https://interactive-rpg-map-vercel.vercel.app/

## Technologies Used
- Visual Studio Code
- HTML
- CSS
- JavaScript
- Leaflet.js
- JSON data files
- Paint (where I made the Pins)
- Wonderdraft (where I made the map)

## Features
- Interactive fantasy world map
- Custom location pins and icons
- Left sidebar with filters by location type
- Right sidebar with location lore and details
- Dynamic loading of locations from JSON files
- Keyboard and click controls to close panels

## How to Run
This project is static and does not require a build step.

1. Open the project folder in a code editor.
2. Start a local server, since the map loads JSON files with `fetch`.
3. Open `index.html` in your browser through that local server.

If you use VS Code, the Live Server extension is a simple option!
